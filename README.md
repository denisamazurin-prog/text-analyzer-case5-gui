markdown# Текстовый анализатор (Кейс №5)

Десктопное приложение на Python для автоматического подсчета статистики текстовых файлов с адаптивным кодировочным конвейером и графическим интерфейсом.

## Функционал
* Автоматический подбор кодировки (`UTF-8`, `CP1251`, `UTF-16`, `Latin-1`).
* Подсчет символов, строк, общего числа слов и уникальных лексем.
* Извлечение Топ-5 самых частых слов с помощью регулярных выражений.
* Вывод результатов во всплывающее графическое окно с прокруткой (`ScrolledText`).
* Автоматическое сохранение отчета в файл `*_stats.txt`.

## Запуск
```bash
python main.py
```
